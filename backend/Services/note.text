Service tự tạo
Service dùng để xử lý Logic là bộ não nghiệp vụ

👉 Nhiệm vụ
Xử lý logic chính
Validate
Gọi DB
Mapping Model → DTO

📌 Ví dụ
public List<ProductResponse> GetAll()
{
    return _context.Products
        .Select(p => new ProductResponse {
            Id = p.Id,
            Name = p.Name,
            CategoryName = p.Category.Name
        })
        .ToList();
}
